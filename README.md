# wx-echarts

## 安装依赖（推荐使用yarn）

安装 yarn

```
sudo npm install yarn -g
```

安装依赖

```
yarn install 

or 

yarn
```

## 开发

```
yarn dev:dev:mp-weixin

or

npm run dev:mp-weixin
```

## 构建

测试小程序线上版本构建命令：

```
yarn build-test:mp-weixin

or

npm run build-test:mp-weixin
```

正式小程序线上版本构建命令：

```
yarn build-production:mp-weixin

or

npm run build-production:mp-weixin
```
